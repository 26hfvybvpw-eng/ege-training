Источники заданий для тренажёра ЕГЭ
===================================

Каждый файл в этой папке содержит задания одного типа.
Пустые строки и строки, начинающиеся с #, игнорируются.

При запуске программа читает все поддерживаемые файлы и
преобразует их во внутренний формат заданий. Чтобы добавить
новые задания, достаточно отредактировать соответствующий файл.


stress.txt — ударения
---------------------
Одно слово на строку. Ударная гласная обозначается заглавной буквой.

Примеры:
    каталОг
    квартАл
    тОрты

Вопрос формируется автоматически:
    «Куда падает ударение в слове «каталог»?»


paronyms.txt — паронимы
-----------------------
Формат: вопрос|ответ

Пример:
    абонемент|абонент


orthography.txt — орфография
----------------------------
Формат: вопрос|ответ

Пропуски обозначаются двумя точками (..).

Примеры:
    к..саться|о
    з..ря|а


nn.txt — удвоение буквы «н»
---------------------------
Формат: вопрос|ответ

Пример:
    ветре..ый|н


ne_ni.txt — написание «не» и «ни»
---------------------------------
Формат: вопрос|ответ

Пример:
    (не)друг|не


punctuation.txt — пунктуация
-----------------------------
Формат: вопрос|ответ

Файл может быть пустым — программа продолжит работу.

Пример:
    Вставьте запятую|,
